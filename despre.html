Sărățica Nouă

Sărățica Nouă este amplașată in Raionul Leova


Satul Sărăţica Nouă este o localitate in Raionul Leova situata la latitudinea 46.5724 longitudinea 28.5483 si altitudinea de 79 metri fata de nivelul marii. Aceasta localitate este in administrarea or. Leova. Conform recensamintului din anul 2004 populatia este de 760 locuitori. Distanța directă pîna în or. Leova este de 34 km. Distanța directă pîna în or. Chişinău este de 55 km.Primarul este Raevsci Mihail.În el se află multe locuri turistice ca Biserica ,,Sfîntul Nicolae",Statuia ,,Maica îndurerată",lîngă el se află valul lui Traian.
